# Roadmap

This document outlines the future development plans for Personaut.

## Short-term Goals

### 1. User Documentation
-   ✅ Basic Setup Guide
-   ✅ Architecture Overview
-   ⬜ Detailed User Guide for Personas
-   ⬜ Video Tutorials

### 2. Performance Optimization
-   Optimize Webview load time.
-   Reduce bundle size.
-   Improve AI response streaming latency.

### 3. User Feedback Loop
-   Integrate mechanism for users to rate AI responses.
-   Implement telemetry (optional/opt-in) to track error rates.

## Long-term Vision

### 1. Enhanced UI Refactoring
-   Modularize the Webview components further.
-   Implement a strict Design System.
-   Add comprehensive UI tests (Storybook or similar).

### 2. Advanced Persona Features
-   **Persona Evolution**: Personas that "remember" past interactions and evolve their feedback style.
-   **Custom Personas**: Allow users to share and import persona definitions.

### 3. Multi-Agent Collaboration
-   Enable multiple personas to discuss a feature amongst themselves in a "room".
-   "UX Researcher" agent that synthesizes feedback from multiple personas automatically.

### 4. Direct IDE Integration
-   Deeper integration with VS Code APIs (e.g., Code Actions, diagnostics).
-   "Fix it for me" buttons directly in the feedback view.
