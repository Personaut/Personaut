# Personas

The core feature of Personaut. Define user personas with specific backgrounds, technical levels, and personalities.

## Overview
Personas allow you to simulate different types of users interacting with your application. Each persona has a unique voice, set of biases, and technical proficiency.

## Usage
1.  Navigate to the **Personas** tab.
2.  Click **Create Persona**.
3.  Fill in the details:
    - **Name**: e.g., "Sarah the Senior Dev"
    - **Role**: e.g., "Software Engineer"
    - **Goals**: What they want to achieve.
    - **Backstory**: Their history and context.
4.  Use the "Generate" button to have AI create a detailed backstory automatically.
5.  Save the persona.
