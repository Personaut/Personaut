# Build Mode (Planning Phase)

An agentic workflow where Personaut guides you through the product planning life cycle.

## Overview
Build Mode transforms Personaut into a Product Manager and UX Designer. It uses multi-agent workflows to:
1.  **Refine Ideas**: Analyze and improve your initial product concept.
2.  **Define Personas**: Generate detailed user personas based on demographics.
3.  **Interview Users**: Simulate interviews with AI agents acting as your users to gather feature requests.
4.  **Create Stories**: Generate prioritized user stories with acceptance criteria.
5.  **Design**: Produce application flows and key screen specifications.

## Usage
1.  Switch to **Build** mode in the sidebar.
2.  Start a new project or load an existing one.
3.  Progress through the 5 stages:
    - **Idea**: Define the core concept.
    - **Users**: Generate and select your target audience.
    - **Features**: Simulate interviews to discover high-impact features.
    - **Stories**: Translate features into technical user stories.
    - **Design**: Visualize user flows and screen layouts.
4.  All data is persisted in the `.personaut/{project}/planning/` directory.
