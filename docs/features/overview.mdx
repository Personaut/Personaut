# Features Overview

Personaut provides a suite of tools to help you build user-centric software.

- [**Personas**](personas.mdx) - Create and simulate user personas.
- [**Feedback Loop**](feedback.mdx) - Get AI-driven feedback on your work.
- [**Build Mode**](build-mode.mdx) - Agentic coding assistance.
- [**Settings**](settings.mdx) - Configure providers and preferences.
