# Settings

Configure your AI providers and extension preferences.

## Providers
Personaut supports multiple AI backends:
-   **Google Gemini**: Recommended for speed and multimodal capabilities.
-   **OpenAI**: Standard GPT-4/4o support.
-   **AWS Bedrock**: Enterprise-grade access to Claude (Sonnet, Opus) and other models.
-   **Native IDE**: Uses the VS Code `lm` API (requires GitHub Copilot Chat).

## Configuration
-   **API Keys**: Keys are stored securely in VS Code's secret storage, not in settings files.
-   **System Prompts**: Customize the base behavior of the agent to fit your specific workflow.
-   **Rate Limits**: Set token usage limits to control costs.
